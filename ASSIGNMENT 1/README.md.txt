# 📘 Full Stack Lab - Assignment 1

This repository contains my first lab assignment for Full Stack Development, created using only HTML.

##  Projects Included

### 1️ HTML Table Assignment

* Created a complete HTML table
* Used various HTML tags such as table, tr, td, th, thead, tbody, etc.
* Demonstrates understanding of basic HTML structure and table formatting

### 2️ Cafe CCD Web Page

* Designed a simple Cafe Coffee Day (CCD) webpage using only HTML
* Includes basic layout structure, text formatting, images, and links
* Focused on practicing core HTML tags and page structure

## 🛠️ Technologies Used

* HTML5

##  Purpose

The goal of this assignment was to build a strong foundation in HTML by practicing different tags and creating simple web page structures without using CSS or JavaScript.
